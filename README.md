# ⚡ GOODFPS - High Performance Game Booster (V1.0) 💎

> **"Transforme sua batata em uma máquina de elite com um único comando."**

O **GOODFPS** é um otimizador de processos via terminal projetado para extrair o máximo de frames por segundo (FPS) em qualquer hardware. Ele atua diretamente na prioridade da CPU, variáveis de ambiente da GPU e limpeza de texturas desnecessárias.

---

### 🚀 O que o GOODFPS faz por você?

* **🎯 Auto-Detection:** Monitora o sistema em tempo real e ativa o boost instantaneamente assim que detecta o jogo aberto.
* **🥔 Potato Mode (Roblox):** Remove texturas pesadas e sombras complexas para garantir fluidez máxima em PCs fracos.
* **🔓 FPS Unlocker:** Quebra a trava nativa de 60 FPS, permitindo que seu hardware entregue tudo o que pode.
* **🔋 Power Plan Injection:** (Windows) Força o plano de energia de "Desempenho Máximo" oculto do sistema.
* **📊 Live Monitoring:** Interface hacker que mostra FPS, latência e consumo de CPU durante a gameplay.

### 📥 Guia de Instalação

#### 📱 No Termux (Android)
```bash
pkg update && pkg upgrade -y
pkg install python git -y
git clone [https://github.com/Anonymous40443/GOODFPS.git](https://github.com/Anonymous40443/GOODFPS.git)
cd GOODFPS
python GOODFPS.py

### 🐧 No Linux (Ubuntu / Kali / Debian)
Bash

sudo apt update && sudo apt install python3 git -y
git clone [https://github.com/Anonymous40443/GOODFPS.git](https://github.com/Anonymous40443/GOODFPS.git)
cd GOODFPS
sudo python3 GOODFPS.py

### 🪟 No Windows (CMD / PowerShell)

    Python: Instale o Python e marque a caixa "Add to PATH".

    Execução: Abra o CMD ou PowerShell como Administrador.

    Comando:

PowerShell

cd C:\Caminho\Ate\A\Pasta\GOODFPS
